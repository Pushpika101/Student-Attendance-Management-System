AttendPro – Student Attendance Management System

📌 Overview

AttendPro is a desktop-based Student Attendance Management System developed using VB.NET and MySQL.
It is designed to replace traditional paper-based or Excel-based attendance tracking with a more efficient, accurate, and automated solution.

This system is especially suitable for small to medium-sized educational institutions.

⸻

🎯 Features

* 🔐 User Authentication
    * Secure login for Admin and Lecturers
* 👨‍🎓 Student Management
    * Add, update, delete student records
    * Store student details and course enrollment
* 📚 Course & Lecture Management
    * Manage courses and lecturers
    * Assign students to courses
* 📝 Attendance Tracking
    * Mark attendance (Present / Absent / Late)
    * Daily attendance recording
* 📊 Report Generation
    * Attendance percentage calculation
    * Absentee reports
    * Printable reports
* ⚡ Real-Time Data Access
    * Instant updates for attendance records
* 🚨 Alerts (Optional Enhancement)
    * Identify students with low attendance

⸻

🏗️ System Architecture

* Frontend: VB.NET (Windows Forms)
* Backend: MySQL Database
* Platform: Windows (Visual Studio 2022)

⸻

🗄️ Database Structure

Main Tables:

* Student
* Course
* Lecturer
* Attendance
* User (Login System)

⚙️ Installation & Setup

1. Clone the repository:
   git clone https://github.com/Pushpika101/Student-Attendance-Management-System.git
2. Open the project in Visual Studio 2022
3. Setup MySQL Database:
    * Create a database (e.g., attendpro_db)
    * Import provided .sql file (if available)
4. Configure Database Connection:
    * Update connection string in the project
5. Run the project:
    * Press Start (F5) in Visual Studio

⸻

🚀 Usage

1. Login as Admin or Lecturer
2. Add Students, Courses, and Lecturers
3. Mark attendance for each session
4. Generate reports and analyze attendance
